# 🏔️ Gipfelsturm – Interaktiver Geburtstagsgutschein

Ein interaktiver Gutschein für Johannes: Eine 2-Tages-Bergtour auf die Zugspitze über den Stopselzieher-Klettersteig.

## 🚀 Setup (GitHub Pages)

### 1. Repository erstellen

- Erstelle ein neues Repository auf GitHub
- **Name:** Wähle etwas Unauffälliges, z.B. `birthday-surprise` oder `party-app` (nicht `zugspitze-gutschein`, falls Johannes dein GitHub findet!)
- **Sichtbarkeit:** Public (nötig für GitHub Pages im Free-Tier)

### 2. Dateien hochladen

Lade folgende Dateien in das Root-Verzeichnis des Repos:

```
birthday-surprise/
├── index.html              ← Die komplette App
├── img/
│   └── zugspitze.jpg       ← Dein Zugspitze-Bild (umbenennen!)
└── README.md               ← Diese Datei (optional)
```

**Wichtig:** Benenne dein Bild in `zugspitze.jpg` um und lege es in einen Ordner namens `img/`.

### 3. GitHub Pages aktivieren

1. Gehe zu **Settings** → **Pages**
2. Unter "Source" wähle **Deploy from a branch**
3. Branch: **main**, Ordner: **/ (root)**
4. Klick auf **Save**
5. Nach 1–2 Minuten ist deine Seite live unter: `https://DEIN-USERNAME.github.io/birthday-surprise/`

### 4. Link teilen

Schicke Johannes den Link – fertig! 🎉

## 🎮 Wie es funktioniert

- **Party-Modus:** 5 verrückte Challenges, die Johannes vor Publikum erledigen muss. Jede Challenge wird von der Jury per PIN bestätigt.
- **Solo-Modus:** 5 Quizfragen (Thema: Hobby Horsing 😄) – erst wenn alle beantwortet sind, wird der Gutschein enthüllt.
- **PIN für die Jury:** `4321`
- Bei falschem PIN: Dramatische Alarm-Animation + Fake-Selbstzerstörung des Gutscheins
- Bei falschen Quiz-Antworten: Eskalierende Sprüche 😂

## 📝 Anpassen

Alles ist in der `index.html` – du kannst Texte, Challenges, Quiz-Fragen, PIN und Farben direkt in der Datei ändern.
